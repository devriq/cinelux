# Cinelux

A movie catalog website connected to the TMDB API.



## Tech Stack

**Client:** React, TailwindCSS


## Deployment

To deploy this project run the following code inside the project root directory

```bash
  npm run dev
```
## Features

- Movie detail page
- Mobile responsive layout
